# SistemaCadastroMedilab

Projeto em Java desenvolvido para estudo de lógica de programação, Programação Orientada a Objetos (POO) e estruturas de dados.

## Descrição
O SistemaCadastroMedilab é uma aplicação em Java executada no console, que simula o cadastro de pacientes de um laboratório de exames médicos.

## Funcionalidades
- Menu interativo no console
- Cadastro de pacientes (nome, idade e CPF)
- Listagem de pacientes cadastrados
- Armazenamento em memória utilizando ArrayList

### Tecnologias utilizadas
- Java
- Programação Orientada a Objetos (POO)
- ArrayList
- Scanner
- Git e GitHub

## Objetivo do projeto
Este projeto tem como objetivo praticar conceitos fundamentais de Java e servir como base para evolução futura em backend, incluindo CRUD completo, banco de dados e APIs REST com Spring Boot.
